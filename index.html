El origen

La Inteligencia Artificial siempre hay que vislumbrarla como un amplio campo que abarca muchas disciplinas. Dentro de esta existe el Machine Learning, que se fundamenta en el aprendizaje a partir de datos. Y a su vez, esta última incluye el Deep Learning que emplea modelos inspirados en la estructura y funcionamiento de las neuronas.

Las Redes Neuronales

Una Red Neuronal está formada por dos o más capas. Cada una de estas capas se compone de una o más neuronas.

Como mínimo son necesarias dos capas en una Red Neuronal: una capa de entrada y otra de salida.

La capa de entrada tendrá tantas neuronas como datos de entrada se vayan a proporcionar a la Red Neuronal.

La capa de salida tendrá tantas neuronas como respuestas se quieran obtener de la red neuronal.

Entre las capas de entrada y salida, es posible añadir todas las capas que consideremos necesarias, cada una con la cantidad de neuronas que nos parezca más adecuada.

//Ejemplo// Si buscamos saber si una planta está sana o no, en base a su incremento de crecimiento, color RGB y longitud de las hojas.

Un modelo basado en red neuronal, tendrá una capa de entrada con cinco neuronas. Cada una para recibir cada dato de entrada.

La capa de salida tendrá una única neurona, ya que buscamos una respuesta binaria (Sí o No está sana).

Entre la capa de entrada y la capa de salida, opcionalmente, se podrían colocar otras capas con neuronas. Por ejemplo tres capas: * Una primera, con 9 neuronas. * Una segunda, con 18 neuronas. * Una tercera, con 3 neuronas.

Las salidas de cada neurona, está conectada como entrada de todas las neuronas de la siguiente capa.

Principios de funcionamiento

Cada neurona aplica una función de Machine Learning sencilla, a los valores que recibe de entrada para producir una única salida. Las salidas de cada neurona de una capa se combinan como entradas de cada neurona de la siguiente capa.
